About

This template is a work in progress layout for many sites I have made, and since it's pretty nice, I decided to release it to GitHub. I am slowly pushing more code out, as some is still very messy and hard to read, so some parts are unfinished. You may find some very ugly looking css elements, including input boxes, buttons etc, they will be fixed in a future release.

Source code

The source code for this template is available at https://github.com/jono292/material-webpage-template. You are free to use this code and modify it as you wish, as stated in the Apache 2.0 license, but please credit me.
